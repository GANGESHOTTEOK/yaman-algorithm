### git 활용법

```shell
// 1. repo 클론하기
git clone https://github.com/GANGESHOTTEOK/yaman-algorithm.git

// 2. 자신의 branch 생성 및 이동
git checkout -b 이름

// 3. 코드 및 md 작성 후 스테이징

// 4. 커밋
git commit -m "코드 작성"

// 5. 온라인 repo에 push

// 6. 깃허브 repo 페이지에서 pull request 하기 (매번 할 필요는 없음)
```
